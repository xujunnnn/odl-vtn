RPC Definitions

get-manager-version

  • Description: Return the version information of the VTN Manager.
    • container: input
    • container: output
        • leaf: api-version
          • description: A numerical value which represents API version of the VTN Manager. API version is a numerical value equal to or greater than 1, and it is incremented if API of the VTN manager is changed. Current VTN Manager returns 2 as API version.
          • type: uint32
        • container: bundle-version
          • description: Describes OSGi bundle version of the VTN Manager. Note that this container will be omitted if OSGi bundle version is not available.
            • leaf: major
              • description: The major component of the OSGi bundle version identifier.
              • type: (urn:ietf:params:xml:ns:yang:1)uint32
            • leaf: minor
              • description: The minor component of the OSGi bundle version identifier.
              • type: (urn:ietf:params:xml:ns:yang:1)uint32
            • leaf: micro
              • description: The micro component of the OSGi bundle version identifier.
              • type: (urn:ietf:params:xml:ns:yang:1)uint32
            • leaf: qualifier
              • description: The qualifier component of the OSGi bundle version identifier. This field is omitted if the qualifier component is not set in the OSGi bundle version identifier.
              • type: (urn:ietf:params:xml:ns:yang:1)string
