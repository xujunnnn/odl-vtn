Type Definitions

  • vtn-openflow-version

    • Description: Describes the OpenFlow protocol version number.
    • Base type: enumeration

Groupings

  • vtn-port-info

    • Description: Generic information about a switch port managed by the VTN Manager.
      • leaf: id
        • description: The unique identifier for the switch port.
        • type: (urn:opendaylight:inventory)node-connector-id
      • leaf: name
        • description: Human readable name of the switch port.
        • type: (urn:ietf:params:xml:ns:yang:1)string
      • leaf: enabled
        • description: Determine whether the switch port is enabled or not.
        • type: (urn:ietf:params:xml:ns:yang:1)boolean
      • leaf: cost
        • description: The cost of the link for transmitting a packet from this port.
        • type: (urn:ietf:params:xml:ns:yang:1)int64
      • list: port-link
        • description: A list of inter-switch links.
        • key definition: link-id
          • leaf: link-id
            • description: The identifier of the inter-switch link.
            • type: (urn:TBD:params:xml:ns:yang:network-topology)link-id
          • leaf: peer
            • description: The identifier of the peer switch port of the link.
            • type: (urn:opendaylight:inventory)node-connector-id

  • vtn-node-info

    • Description: Generic information about a node managed by the VTN Manager.
      • leaf: id
        • description: The unique identifier for the switch.
        • type: (urn:opendaylight:inventory)node-id
      • leaf: openflow-version
        • description: Estimated OpenFlow protocol version currently used on the secure channel.
        • type: vtn-openflow-version
      • list: vtn-port
        • description: A list of physical switch ports that belong this node.
        • key definition: id
        • uses: vtn-port-info
          • refines:
          • leaf: id
            • description: The unique identifier for the switch port.
            • type: (urn:opendaylight:inventory)node-connector-id
          • leaf: name
            • description: Human readable name of the switch port.
            • type: (urn:ietf:params:xml:ns:yang:1)string
          • leaf: enabled
            • description: Determine whether the switch port is enabled or not.
            • type: (urn:ietf:params:xml:ns:yang:1)boolean
          • leaf: cost
            • description: The cost of the link for transmitting a packet from this port.
            • type: (urn:ietf:params:xml:ns:yang:1)int64
          • list: port-link
            • description: A list of inter-switch links.
            • key definition: link-id
              • leaf: link-id
                • description: The identifier of the inter-switch link.
                • type: (urn:TBD:params:xml:ns:yang:network-topology)link-id
              • leaf: peer
                • description: The identifier of the peer switch port of the link.
                • type: (urn:opendaylight:inventory)node-connector-id
