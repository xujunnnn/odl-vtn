vtn-mapping

Base Information

prefix vmapping
namespace urn:opendaylight:vtn:impl:mapping
revision 2015-10-01
description The internal module to maintain mappings between virtual nodes and physical network elements. Application other than VTN Manager must not use models defined in this module.
yang-version 1
imports l2:opendaylight-l2-types 2013-08-27; yang:ietf-yang-types 2013-07-15; vtype:vtn-types 2015-02-09;

Groupings Summary

Name Description

vtn-map-reference-set Describes a set of virtual network mappings associated with physical network element.

vtn-map-reference-list Describes a set of virtual network mappings.

Groupings

  • vtn-map-reference

    • Description: Describes a virtual network mapping associated with physical network element.
      • leaf: identifier
        • description: Describes an identifier for a physical network element.
        • type: (urn:ietf:params:xml:ns:yang:1)string
      • leaf: mapping
        • description: Describes a virtual network mapping. The value of this field contains the type of mapping and the location of the virtual node.
